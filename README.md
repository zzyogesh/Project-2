# 🌾 Farmer Guider App

## 📋 Crop Production
A comprehensive machine learning application designed to assist farmers in making informed decisions about crop selection and fertilizer recommendations based on soil and environmental parameters.

## 📝 Project Description
The Farmer Guider App is an intelligent web application that leverages machine learning models to:
- **Crop Recommendation**: Suggest suitable crops based on soil nutrients (Nitrogen, Phosphorus, Potassium), temperature, humidity, pH level, and rainfall
- **Fertilizer Recommendation**: Recommend appropriate fertilizers based on soil composition and crop type
- User-friendly interface for easy access to recommendations
- Data-driven insights to optimize agricultural productivity

## 🛠️ Technology Used
- **Backend**: Python Flask (Web Framework)
- **Machine Learning**: Scikit-learn
- **Frontend**: HTML5, CSS3
- **Data Processing**: Pandas, NumPy
- **Dataset**: Agricultural crop and fertilizer datasets

## 🔄 Machine Learning Workflow
1. **Data Collection**: Gathered agricultural datasets with crop and fertilizer parameters
2. **Data Preprocessing**: Cleaned and normalized data for model training
3. **Feature Engineering**: Selected relevant features for prediction accuracy
4. **Model Training**: Trained multiple ML algorithms and selected best performers
5. **Model Evaluation**: Validated models using accuracy metrics
6. **Deployment**: Integrated models into Flask web application
